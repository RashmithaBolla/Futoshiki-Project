# Futoshiki-Project
This project describes the creation of an interactive digital Futoshiki game. The game features multiple levels of difficulty and puzzle generation, utilizing a 4x4 grid with specific numbers assigned to each row and column, adhering strictly to inequality restrictions, multiple difficulty tiers, and an automatic timer. This game is constructed using Python and offers a colourful, appealing, and creative user interface. The libraries used in this project are pygame, time, math and random. This project contributes to both educational entertainment and technological innovation. It also improves critical thinking and logical deduction. The outcome of this project is a completely interactive, operational Futoshiki game capable to check and verify riddles.
